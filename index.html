🧪 用法 Usage

展示 1–2 个常见脚本与调用方式（训练 / 推理 / 评估）。

训练 Train

# 示例：训练
python tools/train.py \
  --data ./data \
  --epochs 100 \
  --batch-size 32 \
  --lr 2e-4 \
  --out ./logs

推理 / 重建 Inference

# 示例：推理 / 重建
python tools/eval_reconstruct.py \
  --ckpt ./checkpoints/best.pt \
  --split test \
  --num-points 2048

Python API（可选）

from your_repo import Model

model = Model.from_pretrained("checkpoints/best.pt").to("cuda")
points = model.reconstruct(tokens, steps=100)
